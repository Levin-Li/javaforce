To compile:

Must compile on a 64bit host (both 32/64bit images are created).

Required packages : g++ openjdk-8-jdk ant libx11-dev libfuse-dev libpam0g-dev
  libavcodec-dev libavformat-dev libavutil-dev libswscale-dev
To build the 32bit image: libc6-dev:i386 libx11-dev:i386 libpam0g-dev:i386

export JAVA_HOME=/usr/lib/jvm/java-8-openjdk-amd64
